# Collapsible

Toggles elements between `height: 0` and `height: 'auto'`.

This is a very low-level component to work with so you can construct it every which way you want.

This component uses [`render-props`](https://reactjs.org/docs/render-props.html); inspect the source of the example to get a feel for how it works.

## Custom overflow and height

`<Collapsible>` does not apply an `overflow` to open elements, to set the `overflow` property you can manually add it to the `style`
or use a nested block element for composition with custom styles.

<Playground>
  <Collapsible open>
    {({ isOpen, toggle, open, close, style }) => (
      <Segment.Group>
        <Segment.Header aria-expanded={isOpen} onClick={() => toggle()}>
          Toggle
          <Icon name={isOpen ? 'keyboard_arrow_down' : 'keyboard_arrow_up'} />
        </Segment.Header>
        <Segment
          style={{
            ...style,
            overflow: isOpen ? 'scroll' : style.overflow,
            height: isOpen ? 200 : style.height
