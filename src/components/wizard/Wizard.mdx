# Wizard

Allows you to create a wizard component.
Every child is passed a wizard object that contains the following:

* currentStep: integer
* setSteps: function
* next: function
* previous: function
* endIndex: integer

The Steps component implements the basic functionality of only showing the currently selected step.
The Navigation component implements 2 basic navigation buttons

Children render props can be used or normal child elements.

## Basic usage

<Playground>
  <Wizard>
    <Wizard.Steps>
      <div>
        <h3>Step 1</h3>
      </div>
      <div>
        <h3>Step 2</h3>
      </div>
      <div>
        <h3>Step 3</h3>
      </div>
    </Wizard.Steps>
    <Wizard.Navigation />
  </Wizard>
</Playground>
