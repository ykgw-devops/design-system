# Wizard

Allows you to create a wizard component.
Every direct child of the Wizard component is passed a wizard object that contains the following:

* currentStep
* currentIndex
* goTo
* steps
* totalSteps
* setSteps
* next
* previous
* isLast
* isFirst

The basic Steps component returns the currently selected step.
The basic Navigation component implements 2 basic navigation buttons.

Children render props can be used or normal child elements.

## Basic usage

## Base components

```
const Steps = ({ children = [], wizard }) => {
  const { setSteps, currentStep } = wizard

  useEffect(() => {
    setSteps(castArray(children))
  }, [])

  return currentStep
}
```
```
const Navigation = ({ wizard }) => {
  const { previous, next, isLast, isFirst } = wizard

  return (
    <div>
      <Button disabled={isFirst} onClick={previous}>Previous</Button>
      <Button disabled={isLast} style={{ marginLeft: '5px'}} onClick={next}>Next</Button>
    </div>
  )
}
```

## Custom Components

<Playground>
  <Wizard>

        </div>,

      {
        ({ isFirst, isLast, next, previous }) => (

            <Button disabled={isFirst} kind='warning' onClick={() => previous()}>

            </Button>
            <Button disabled={isLast} kind='warning' style={{ marginLeft: '5px'}} onClick={() => next()}>
              Next
            </Button>
          </>
        )
      }
    </div>
  </Wizard>
</Playground>



## Advanced usage

### All in a single component

Since each component gets the wizard object injected we don't need to split up the steps and navigation, we can just as well put everything in one component.

<Playground>
  <Wizard>
    <div>
      {({ currentIndex, currentStep, setSteps, totalSteps, next, previous, isFirst, isLast }) => {
        const steps = [
          <div>
            <h3>Step 1</h3>
          </div>,
          <div>
            <h3>Step 2</h3>
          </div>,
          <div>
            <h3>Step 3</h3>
          </div>
        ]

        useEffect(() => {
          setSteps(steps)
        }, [])

        return (
          <Segment.Group>
            <Segment.Header>
              Step {currentIndex + 1} of {totalSteps}
            </Segment.Header>
            <Segment>
              { currentStep }
            </Segment>
            <Segment.Footer>
              <Button disabled={isFirst} kind='success' onClick={() => previous()}>
                Previous
              </Button>
              <Button disabled={isLast} kind='success' onClick={() => next()} style={{ marginLeft: '5px'}}>
                Next
              </Button>
            </Segment.Footer>
          </Segment.Group>
        )
      }}
    </div>
  </Wizard>
</Playground>

Through render props this gets quite bulky so for this a custom component might be better like so.

```
const SingleComponent = ({ wizard, children }) => {
  const { currentIndex, currentStep, setSteps, totalSteps, next, previous, isFirst, isLast } = wizard

  useEffect(() => {
    setSteps(children)
  }, [])

  return (
    <Segment.Group>
      <Segment.Header>
        Step {currentIndex + 1} of {totalSteps}
      </Segment.Header>
      <Segment>
        { currentStep }
      </Segment>
      <Segment.Footer>
        <Button disabled={isFirst} kind='warning' onClick={() => previous()}>
          Previous
        </Button>
        <Button disabled={isLast} kind='warning' onClick={() => next()} style={{ marginLeft: '5px'}}>
          Next
        </Button>
      </Segment.Footer>
    </Segment.Group>
  )
}
```
